When Biodiversity Becomes Digital


This project explores how the meaning of biodiversity can change depending on the digital tools that are used to understand environmental data. Each phrase is a short description of a different tool. These one-sentence summaries emerged from qualitative research into the TNFD Tools Catalogue. The digital tools can be filtered through 4 categories:


Measuring - detection tools to make biodiversity observations (e.g. satellite imagery, sensors, eDNA)

Cataloging - databases that archive and store biodiversity information (e.g. global indices, species lists, geospatial registers)

Assessing - interpreting data to generate biodiversity insights (e.g. analytical software, data modeling, risk assessments)

Monetizing - assigning financial structures to biodiversity (e.g. funding, trade, offsetting, investment strategies)



Created at the MIT Climate and Sustainability Consortium by Yildiz Derinkok, with mentorship from Dr. Michelle Westerlaken.


This website is an educational project that provides one-sentence summaries of tools listed in the TNFD Tools Catalogue. These summaries are interpreted based on publicly available information, primarily from the tools' websites. While we aim for accuracy, the descriptions may not capture every nuance or feature and should not be considered official representations. Each summary includes a link to the original source for reference.
